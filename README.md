# 3D-Light-Portfolio

3D Light Portfolio — a minimal, responsive single-page portfolio built with HTML, CSS and JavaScript. It showcases a 3D light/scene effect that can be used as a stylish landing section for personal portfolios and small projects.

---

![Preview of the 3D Light Portfolio](Screenshot%202025-10-05%20145022.png)

---

## Features

- Lightweight: pure HTML, CSS and vanilla JavaScript (no frameworks).
- 3D-styled light/scene effect for a modern portfolio landing.
- Responsive layout — works on desktop and mobile screens.
- Easy to customize: colors, text and the 3D effect settings are in the CSS and JS files.

## Demo

Open index.html in a browser to view the portfolio locally:

1. Clone the repo:

   git clone https://github.com/BinaryVortex/3D-Light-Portfolio.git

2. Open the project folder and open `index.html` in your web browser.

   - On most systems you can double-click the file or run `open index.html` (macOS) / `xdg-open index.html` (Linux).

If you want a local server (recommended for some browser feature compatibility):

- Python 3: `python -m http.server 8000` then visit `http://localhost:8000`.

## File structure

- `index.html` — main page and markup.
- `styles/` or inline CSS — styling and 3D effect rules (if present in the repo).
- `script.js` or inline JS — interactive behaviour and animations (if present).
- `Screenshot 2025-10-05 145022.png` — project preview used in this README.

(Depending on how this repo is organized the CSS/JS may be inline inside `index.html`.)

## Customization

- Text: edit the content inside `index.html` to add your name, description, links and contact details.
- Colors & lighting: open the CSS file or CSS block in `index.html` and change color values and shadows.
- Effects & animation: edit the JavaScript in the repository to tweak animation timing or interaction.
